FAQ

Does my team have to change how they work?

No. Pulse reads what's already in Jira — no standup prompts, no forms, no templates.

What data leaves my site?

Issue summaries, statuses, dates, and sprint metadata for the projects you select are sent to Anthropic's API to write the digest; the finished digest goes to the channels you configure. Comments, descriptions, and Confluence excerpts are read only if an admin opts in. Full details: privacy & data handling.

Where do I see digests inside Jira?

The Pulse Hub (Apps → Pulse) shows the latest portfolio view with health, trends, and per-project detail; every project also has a Pulse Digest page.

Something broke?

Set a failure-alert email in the admin page and Pulse tells you when a scheduled run fails. For anything else, use the form below — we respond within 48 hours.
